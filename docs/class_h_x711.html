◆ is_ready()

bool HX711::is_ready ( )

check if HX711 is ready from the datasheet: When output data is not ready for retrieval, digital output pin DOUT is high. Serial clock input PD_SCK should be low. When DOUT goes to low, it indicates data is ready for retrieval.
